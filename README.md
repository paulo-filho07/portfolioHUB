# PortfolioHUB

## Sobre o Projeto

Este repositório foi desenvolvido com o objetivo de organizar e versionar projetos acadêmicos e pessoais utilizando Git e GitHub, aplicando boas práticas de desenvolvimento, documentação e compartilhamento de código.

O projeto faz parte da atividade prática da disciplina, com foco em versionamento, organização de repositórios e publicação de páginas utilizando GitHub Pages.

---

# Objetivos

- Aprender versionamento com Git;
- Utilizar o GitHub para hospedagem de projetos;
- Organizar projetos acadêmicos e pessoais;
- Aplicar boas práticas de documentação;
- Publicar páginas utilizando GitHub Pages;
- Integrar projetos ao LinkedIn profissional.

---

# Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages
